Visora – Face Recognition System 📌 Project Overview

Visora is a real-time Face Recognition System developed as a Final Year Project by a team of four members. The system captures facial images, trains a recognition model, and identifies registered users in real time through a webcam. The project aims to provide a secure, efficient, and automated identity verification solution using Computer Vision techniques.

🚀 Features Real-time face detection using webcam User face enrollment and dataset creation Face recognition using LBPH algorithm Live recognition with confidence score display Automated user identification User-friendly and responsive web interface Attendance/identity verification support Recognition log management 🛠️ Technologies Used Frontend React.js HTML5 CSS3 JavaScript Backend Python OpenCV Face Recognition Technologies Haar Cascade Classifier LBPH (Local Binary Patterns Histogram) 🎯 Project Objectives Develop a reliable real-time face recognition system. Automate user identification and verification. Improve security compared to traditional password-based systems. Reduce manual effort in attendance and access control processes. 👨‍💻 My Contribution

Improving user experience and application navigation. Testing and debugging frontend functionalities. 📷 System Workflow User registration and face image capture. Dataset generation and model training. Real-time webcam feed processing. Face detection using Haar Cascade. Face recognition using LBPH. Displaying recognized user information. 🔍 Problem Statement

Traditional identification systems such as passwords, ID cards, and manual attendance methods can be insecure, time-consuming, and prone to misuse. Visora addresses these challenges by providing an automated and contactless facial recognition solution.
